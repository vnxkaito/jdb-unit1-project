<h1>ACME banking CLI</h1>

<h2>Project Overview</h2>
<p>This project is a basic banking system with a CLI. The purpose of this project is to explore the different aspects of java programming.</p>

<p>To practice TDD, unit tests will be created before the actual functionalities are implemented.</p>

<h2>Phases</h2>
<h3>Phase 1</h3>
Analyze the requirements.
Draft the entites and classes.
Define the commands and their parameters.
Define the user stories and their corresponding unit tests.
Implement the dialog for all the commands.

<h3>Phase 2</h3>
Create all the unit tests.

<h3>Phase 3</h3>
Create all classes with hard coded data for the entities.

<h3>Phase 4</h3>
Implement the CRUD operations for all entities and remove the hard coded data from the entities classes.
